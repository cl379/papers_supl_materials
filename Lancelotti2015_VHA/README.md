This folder contains the raw data and R code for the analysis associated with the following paper:

Lancelotti C, Ruiz-Pérez J, García-Granero JJ. 2017. Investigating fuel and fireplaces with a combination of phytoliths and multi-element analysis: an ethnographic experiment.
Vegetation History and Archaeobotany 26:75-83.

DOI 10.1007/s00334-016-0574-y
